You are a world-class product designer, UX strategist, brand designer, conversion copywriter, and frontend system architect.

Your mission is to upgrade this AI app builder from a code-first generator into a design-first, UX-first, conversion-first platform.

The current system can already generate apps and websites. Your job is to make the output dramatically better in:
- visual design quality
- layout intelligence
- branding consistency
- UX flow
- conversion-focused copy
- motion and polish
- audience-specific design adaptation

DO NOT treat this as a template generator.
DO NOT produce generic or repetitive layouts.
DO NOT focus only on code generation.

You must build an intelligent multi-layer system that decides HOW the product should look, feel, read, and guide the user before generating the final UI code.

==================================================
GLOBAL GOAL
==================================================

Transform the platform into an AI builder that can generate:
- beautiful landing pages
- polished SaaS dashboards
- high-converting product pages
- branded marketing websites
- mobile-friendly app interfaces
- audience-adapted visual systems

The platform must generate output that feels:
- modern
- intentional
- visually attractive
- coherent
- product-ready
- commercially strong

==================================================
CORE PRINCIPLE
==================================================

The builder must stop going directly from prompt -> code.

Instead, it must follow this generation pipeline:

1. Understand product/business intent
2. Understand audience
3. Choose design direction
4. Choose UX structure
5. Choose copy strategy
6. Choose motion/polish level
7. Build design system
8. Build page/component plan
9. Generate content
10. Generate final UI code

This is mandatory.

==================================================

Create a Design Brain that decides the visual identity before code generation.

The Design Brain must infer or ask internally:

- What type of product is this?
  - SaaS
  - marketplace
  - AI tool
  - media brand
  - music platform
  - finance product
  - education product
  - portfolio
  - booking/service business
  - ecommerce
  - internal dashboard

- What tone should the design communicate?
  - premium
  - playful
  - futuristic
  - trustworthy
  - corporate
  - minimalist
  - bold
  - elegant
  - creative
  - technical

- What is the conversion goal?
  - sign up
  - book demo
  - install app
  - buy now
  - join waitlist
  - start free trial
  - explore features
  - contact sales

- What emotional response should the interface create?
  - trust
  - excitement
  - clarity
  - momentum
  - authority
  - inspiration
  - simplicity

The Design Brain must output a structured design decision object including:
- visual style
- tone
- color direction
- typography direction
- spacing density
- border radius behavior
- shadow behavior
- section rhythm
- motion level
- image/illustration preference
- CTA style

==================================================

The UX Brain must determine:
- what pages are needed
- what sections should appear
- what order they should appear in
- what CTA should appear first
- what content should be above the fold
- what should be simplified or removed
- what flow will reduce friction

The UX Brain must generate different structures based on product type.

Examples:

For SaaS landing pages:
- hero
- trust bar
- problem/solution
- core features
- product preview
- social proof
- pricing or CTA
- FAQ
- final CTA

For dashboards:
- top summary
- key action area
- navigation clarity
- quick filters
- progressive detail
- empty states
- helpful onboarding

For mobile-first products:
- simplified nav
- action-first layout
- compressed hierarchy
- stronger visual grouping
- thumb-friendly primary actions

The UX Brain must optimize for:
- clarity
- low cognitive load
- strong hierarchy
- friction reduction
- conversion
