7. Deploy State Machine
ב-routes/projects/deploy.ts,
החלף את הלוגיקה הישירה ב-state machine:
states: pending → queued → uploading → deployed | failed
transitions: retry עד 3 פעמים עם exponential backoff (2s, 4s, 8s)
שמור את ה-state ב-DB בכל מעבר.
שדר WebSocket event על כל שינוי state.
8. GitHub incremental sync
