🧠 Prompt 1 — יצירת Runtime Manager
You are a senior distributed systems architect.

Design and implement a Runtime Control Plane for a cloud AI app builder platform.

Requirements:
- Manage isolated execution environments per project
- Support start/stop/restart of environments
- Track CPU, RAM, GPU usage per environment
- Support ephemeral environments (auto shutdown after inactivity)
- Provide preview URLs per environment
- Maintain environment lifecycle states: creating, running, idle, stopped, failed

Tech constraints:
- Node.js (TypeScript) or Python backend
- Use Docker containers for isolation
- Use Redis for state and queues
- PostgreSQL for persistence

Output:
1. Architecture diagram (text)
2. Database schema for environments table
3. API endpoints (REST)
4. Worker logic for environment lifecycle
5. Example implementation code (core parts only)
